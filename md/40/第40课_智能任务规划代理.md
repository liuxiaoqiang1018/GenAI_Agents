# 第40课：智能任务规划代理（Taskifier）

## 课程概述

本课构建一个**个性化任务规划系统**：根据用户的历史工作记录分析其工作风格偏好 → 搜索目标任务的相关知识 → 结合风格偏好生成定制化的执行方案。

**核心模式**：工作风格分析 + 知识检索 + 个性化方案生成（三步线性流水线）

## 架构流程

```
用户输入任务 → [风格分析] 读取历史工作记录，LLM分析偏好
                  ↓
            [知识检索] 搜索任务相关步骤和知识
                  ↓
            [方案生成] 结合风格+知识生成个性化执行计划
                  ↓
              输出计划 → 结束
```

## 关键概念标注

| 概念 | 在代码中的位置 | Java 类比 |
|---|---|---|
| **历史记录分析** | `approach_analysis()` — 读取 history/*.txt，LLM 分析工作风格 | 用户画像挖掘 |
| **知识检索** | `task_manifest()` — Tavily 搜索任务步骤 | 搜索引擎集成 |
| **个性化生成** | `result_approach()` — 结合风格+知识生成方案 | 模板+策略模式 |
| **State 传递** | `ApproachState` — plan/style/task/details/history | DTO 数据流转 |

## 本课特点

1. **个性化核心**：不是通用的任务拆解，而是根据"你的工作风格"定制方案
2. **历史记录驱动**：从历史文件中提取工作偏好（类似用户画像）
3. **三步纯线性**：最简洁的 LangGraph 用法，没有条件路由和循环
4. **实用场景**：拖延症/工作流混乱的人，AI 帮你规划适合你的步骤
