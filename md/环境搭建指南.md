# 环境搭建指南

## 创建虚拟环境

```bash
# 进入课程目录（以第1课为例）
cd md/01

# 创建虚拟环境
python -m venv venv

# 激活虚拟环境（Windows）
venv\Scripts\activate

# 激活成功后，命令行前面会出现 (venv) 标识
```

## 安装依赖

```bash
# 在激活虚拟环境后安装所需的包
pip install pydantic-ai-slim[openai] python-dotenv nest_asyncio
```

## 运行代码

```bash
python main.py
```

## 退出虚拟环境

```bash
deactivate
```

## 注意事项

- 每课的 `venv/` 和 `.env` 已在 `.gitignore` 中，不会被提交
- 每次打开新的终端窗口都需要重新激活虚拟环境
- 不同课程的虚拟环境相互独立，包版本互不影响
