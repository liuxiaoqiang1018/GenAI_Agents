# 🍳 家庭食谱推荐

根据您的剩余食材，我为您推荐以下3道简单实用的家常菜：

---

## 一、番茄炒鸡蛋 🥇

**所需食材：**
- 番茄 2个
- 鸡蛋 3个
- 食用油、盐、白糖 适量

**做法步骤：**
1. 番茄洗净切块，鸡蛋打入碗中加少许盐搅散
2. 热锅冷油，倒入蛋液快速翻炒至凝固，盛出备用
3. 锅中再加少许油，放入番茄块翻炒出汁
4. 加入1小勺白糖提鲜，倒入炒好的鸡蛋
5. 加盐调味，翻炒均匀即可出锅

**推荐理由：** 经典下饭菜，操作简单，10分钟搞定！

---

## 二、西兰花炒肉片 🥦

**所需食材：**
- 西兰花 1颗（约300g）
- 猪肉里脊 200g
- 食用油、盐、料酒、生抽、淀粉 适量

**做法步骤：**
1. 西兰花洗净切成小朵，用淡盐水浸泡10分钟后焯水备用
2. 猪肉切片，加入料酒、生抽、淀粉抓匀腌制10分钟
3. 热锅冷油，放入肉片滑炒至变色
4. 加入西兰花快速翻炒
5. 加盐调味，翻炒均匀即可

**推荐理由：** 营养均衡，蛋白质和维生素兼顾！

---

## 三、青椒炒豆腐 🫑

**所需食材：**
- 豆腐 2块
- 青椒 2个
- 食用油、盐、生抽、蒜末 适量

**做法步骤：**
1. 豆腐切成1cm厚片，青椒去籽切块
2. 热锅冷油，放入豆腐片小火煎至两面金黄
3. 加入蒜末爆香，放入青椒翻炒
4. 加生抽和盐调味，少许清水焖煮2分钟
5. 大火收汁即可出锅

**推荐理由：** 植物蛋白丰富，低脂健康，老少皆宜！
