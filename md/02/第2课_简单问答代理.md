# 第2课：简单问答代理（Simple Question-Answering Agent）

**本教程基于原始笔记本 `simple_question_answering_agent.ipynb`，演示最基础的 AI 问答代理。**

## 概述

本课介绍如何用 LangChain 构建一个最简单的问答代理。它接收用户问题，通过预定义的提示模板（Prompt Template）格式化后发给大模型（LLM），返回回答。

这是所有 Agent 的起点——理解"提示模板 + LLM"的基本模式。

## 与第1课的区别

| 特性 | 第1课（上下文感知对话代理） | 第2课（简单问答代理） |
|------|---------------------------|----------------------|
| 框架 | PydanticAI | LangChain |
| 对话历史 | ✅ 有（多轮上下文） | ❌ 无（单轮问答） |
| 工具调用 | ✅ 有 | ❌ 无 |
| 复杂度 | 中等 | 最简 |

## 关键组件

1. **语言模型（LLM）**：调用大模型生成回答
2. **提示模板（Prompt Template）**：定义发给 LLM 的提示结构，包含角色设定和用户问题占位符
3. **Chain（链）**：将提示模板和 LLM 串联，形成 `prompt | llm` 的管道

## Java 类比

```
用 Java 的思维理解这段代码：

PromptTemplate  →  相当于 String.format() 的模板
                    "你是AI助手。用户问题：%s。请回答："

ChatOpenAI      →  相当于一个 HttpClient，调用远程 LLM API

Chain (|)       →  相当于 Java Stream 的管道操作
                    template.format(question) → httpClient.post(prompt) → response

get_answer()    →  相当于 Service 层的方法：
                    public String getAnswer(String question) {
                        String prompt = template.format(question);
                        return llmClient.call(prompt);
                    }
```

## 核心流程（3步）

```
用户提问 "法国的首都是哪里？"
    │
    ▼
┌─────────────────────────────────────────────┐
│ PromptTemplate 拼装完整 Prompt：             │
│                                             │
│ "You are a helpful AI assistant.            │
│  User's question: 法国的首都是哪里？          │
│  Please provide a clear and concise answer:" │
└─────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────┐
│ LLM（大模型）接收完整 Prompt，生成回答         │
└─────────────────────────────────────────────┘
    │
    ▼
返回 "法国的首都是巴黎。"
```

**注意**：这里没有工具调用、没有历史记录——这就是最简单的 Agent 形态，本质上就是一个"格式化提示 + 调用 API"的过程。

## 实现

### 安装依赖

```bash
pip install langchain-openai python-dotenv
```

### 导入并初始化

```python
import os
from dotenv import load_dotenv
from langchain_openai import ChatOpenAI
from langchain_core.prompts import PromptTemplate

load_dotenv()
```

### 初始化语言模型

```python
llm = ChatOpenAI(
    model=os.getenv('MODEL_NAME', 'gpt-4o-mini'),
    base_url=os.getenv('API_BASE_URL'),
    api_key=os.getenv('API_KEY'),
    max_tokens=1000,
    temperature=0,
)
```

### 定义提示模板

```python
template = """
You are a helpful AI assistant. Your task is to answer the user's question to the best of your ability.

User's question: {question}

Please provide a clear and concise answer:
"""

prompt = PromptTemplate(template=template, input_variables=["question"])
```

`PromptTemplate` 的作用很简单：把 `{question}` 替换成用户实际的问题。相当于 Java 中的 `MessageFormat.format()`。

### 创建 Chain（链）

```python
qa_chain = prompt | llm
```

`|` 是 LangChain 的管道操作符（LCEL 语法），表示"先执行 prompt 格式化，再把结果传给 llm"。

### 问答函数

```python
def get_answer(question):
    input_variables = {"question": question}
    response = qa_chain.invoke(input_variables).content
    return response
```

### 使用示例

```python
question = "What is the capital of France?"
answer = get_answer(question)
print(f"Question: {question}")
print(f"Answer: {answer}")
# 输出: Answer: The capital of France is Paris.
```

## 总结

这节课学到的核心概念：

- **PromptTemplate**：所有 Agent 的基础，控制发给 LLM 的内容格式
- **Chain（链）**：LangChain 的核心抽象，通过 `|` 把组件串联
- **invoke()**：执行整个链的方法

这是最简单的 Agent 模式。后续课程会在此基础上逐步增加：
- 对话历史（记忆）
- 工具调用（让 Agent 能执行动作）
- 多步推理（ReAct 模式）
