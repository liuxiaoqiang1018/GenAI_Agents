## 1. LLM 调用：优先用 httpx 直接调用

公益站等第三方 API 的 WAF 会屏蔽 OpenAI SDK，从第 7 课起统一用 httpx：

```python
import httpx

def call_llm(messages, model="gpt-4o-mini"):
    url = f"{API_BASE_URL}/chat/completions"
    headers = {
        "Authorization": f"Bearer {API_KEY}",
        "Content-Type": "application/json",
        "User-Agent": "Mozilla/5.0"
    }
    payload = {"model": model, "messages": messages}
    
    for attempt in range(MAX_RETRIES):
        try:
            resp = httpx.post(url, json=payload, headers=headers, timeout=300)
            data = resp.json()
            choices = data.get("choices")
            if not choices:
                raise ValueError("空响应")
            return choices[0]["message"]["content"]
        except Exception as e:
            if attempt < MAX_RETRIES - 1:
                time.sleep((attempt + 1) * 3)  # 指数退避
            else:
                raise
```

**关键参数**：
- 超时：300 秒（120s 不够）
- 重试：MAX_RETRIES=3 + 指数退避
- 空响应防御：检查 choices 是否存在
- User-Agent：必须是 `Mozilla/5.0`

## 2. 如果仍用 SDK，必须加这两个参数

```python
from langchain_openai import ChatOpenAI

llm = ChatOpenAI(
    model="gpt-4o-mini",
    openai_api_base="https://xxx/v1",
    openai_api_key="sk-xxx",
    use_responses_api=False,           # 必须！否则调 /v1/responses 报错
    default_headers={"User-Agent": "Mozilla/5.0"}  # 必须！否则被 WAF 屏蔽
)
```

## 3. 中文化要求

代码中能写中文的地方必须写中文：
- Prompt 模板内容（不用英文模板）
- 示例数据名称和值（"丰田"不是"Toyota"，"张伟"不是"Alice"）
- DataFrame 列名（"价格"不是"Price"）
- 数值符合中国场景（人民币、L/100km）
- **变量名、函数名、代码结构保持英文不变**

---

## 4. main_debug.py 打印要求

每个阶段必须打印，用 `======` 分隔：
1. 发给 LLM 的完整 Prompt（含工具描述、参数、指令）
2. LLM 的决策输出（JSON 格式的工具调用指令）
3. Tool 执行日志（调用了什么、参数、返回值）
4. 第二次发给 LLM 的 Prompt（含工具结果）
5. LLM 的最终回复

---

## 5. main.py 也要打印关键过程

不需要像 main_debug.py 那么详细，但要能看到：
- 格式化后发给大模型的完整 Prompt
- 大模型返回的原始响应
- 工具调用时的参数和返回结果

---

## 6. PydanticAI 注意事项

- 结果取值：`result.output`（不是 `.data` 或 `.content`）
- 动态 system prompt：用 `@agent.system_prompt` 装饰器，把运行时数据（列名、类型、示例）写进去
- 静态 prompt 会导致 LLM 不知道有什么数据，凭空回答

---

## 7. 文档和环境

- 每课 md 文档必须包含 `pip install` 依赖安装说明
- `.env` 文件统一从 `md/01/.env` 同步，不能用旧版本
- MiniMax 等模型返回 `<think>` 标签需清理：`re.sub(r'<think>[\s\S]*?</think>\s*', '', content).strip()`
- Windows 下 Python 输出中文/emoji 需要 `sys.stdout.reconfigure(encoding='utf-8')`
- API_BASE_URL 必须包含 `/v1` 后缀
