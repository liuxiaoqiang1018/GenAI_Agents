# Memory — AI Agent 学习参考文档

> **给 AI 的提示：** 如果你正在帮我做 AI Agent 相关的开发或学习，请先读完这个文件夹下的所有 md 文件，快速了解我的背景、进度和技术规范。

## 这是什么

这是我学习 AI Agent 开发过程中积累的参考文档，用于：
1. 和任意 AI（Claude、ChatGPT、Gemini 等）对话时快速提供背景上下文
2. 自己学习和开发时随时查阅
3. 团队协作时让新成员快速了解项目约定
